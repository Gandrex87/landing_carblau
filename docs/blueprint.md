# **App Name**: Carblau

## Core Features:

- Dynamic Hero Section: Showcase the brand's unique value proposition with the exact Spanish headline 'No somos una tienda de coches al uso. Y eso es bueno para ti.' and a compelling subheadline that emphasizes personalized car finding.
- Interactive Process Showcase: Present the 'Cómo funcionamos' (How we work) section using an engaging, step-by-step interactive carousel to guide users through the service process.
- Prominent Call-to-Action: Integrate a clear and unmissable call-to-action button or section with the text 'Encuentra tu coche ideal' to prompt users to take the next step.
- Personalized Inquiry Form: A simple and modern form to collect user contact details and initial preferences, designed to initiate the personalized car-finding process efficiently.
- Adaptive Responsive Design: Implement a fully responsive layout to ensure an optimal, consistent, and visually appealing user experience across all device sizes, from mobile phones to large desktop screens.
- Integración de Asesor Interactivo (Interactive Advisor Integration): Replace the generic 'Personalized Vehicle Matchmaker' or standard lead form with a specific integration for an external AI Agent. Create a clean, prominent section or a dynamic modal that embeds (via iframe) or links directly to our existing conversational agent hosted at https://my-frontend-app-1063747381969.europe-west1.run.app/. Frame this tool in the UI not as a generic AI, but as our "Asesor Virtual Exclusivo" (Exclusive Virtual Advisor) designed to understand the user's lifestyle before our human experts take over.

## Style Guidelines:

- Primary color: A sophisticated, luminous blue (#7CB9F5) to evoke trust and modernity, prominent against a dark background.
- Background color: A deep, almost-black with a subtle blue hint (#14181C) to provide a sleek, minimalist canvas that highlights content.
- Accent color: A vibrant, clear aqua-cyan (#58E3F6) to create visual interest and highlight interactive elements and call-to-actions, providing energetic contrast.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly legible appearance, maintaining a consistent minimalist aesthetic throughout the page.
- Utilize clean, line-based vector icons with a monochromatic or dual-tone style, consistent with the minimalist and modern design philosophy.
- Adopt a minimalist and spacious layout with ample negative space to ensure clarity and focus on content, employing a modular grid system for easy readability and responsiveness.
- Implement subtle and smooth animations, particularly for the 'Cómo funcionamos' carousel and hover states, to enhance user engagement without distracting from the core content.
